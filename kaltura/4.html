Introduction to Media Gallery

The course Media Gallery is where all videos that have been published to that specific course live. Unlike My Media, videos that are in Media Gallery are viewable by anyone who has access into that course.

Editing your Media Gallery

Using the Edit option you can give your Media Gallery a description or toggle its options. Media Gallery has four options that you can choose to turn off or on, they are:

  1. Moderate content (media will not appear in media gallery until approved by the gallery manager)
  2. Enable comments in Media Gallery
  3. Keep comments private to Media Gallery.
  4. Display embedded media in this Media Gallery

Approving student videos

If you have enabled the Moderate content option then anybody who does not have instructor access into the course will have to have their video approved by an instructor in order to have their videos show up in the course Media Gallery. You will notice that a “Pending” tab will appear whenever someone who is not an instructor attempts to publish a video into the Media Gallery. It is in this tab that you can approve or reject videos.

Analytics

This option can be used to view general statistics concerning the course Media Gallery. Things like the most frequently view video, average duration views & etc. can be viewed here.

Importing a Media Gallery from another course

The Import option will allow you to import all videos saved in another Media Gallery from a different course. Select the course whose Media Gallery you wish to import and all the videos from there will be copied over into the Media Gallery you are currently in.

Please note that you can only import the entire media gallery from another course. Importing individual videos from another media gallery is not possible

Creating Playlists

You can create playlists of selected videos in your Media Gallery by clicking the + Create Channel Playlists option. For more info check out the playlist section of this job aid.
