My Media Video Options

Once a video has been uploaded into My Media you can click that video to access several options that allow you to modify that video. These options are:

  1. Details
  2. Shares
  3. Actions
    1. Edit
    2. Publish
    3. Analytics
    4. Launch Editor
    5. Delete

Please note that modifying a video in one course will affect that video in every course that it is embedded in. It is recommended that you save a copy of a video if you only want to modify it in a single course.

Details

Will show you the video’s title, tags and the names of all of the courses it has been embedded in.

Share

Will produce html code that will allow you to embed the video into any website. You can customize the start and end times as well as the size of the video player from the available options.

Actions

Are options which will allow you to modify the video in a significant way.

Edit

Please see the section on Video Edit Menu below or read this document for further detail

Publish

Will allow you to publish the video into a course Media Gallery if you have not done so already. You can also unpublish the video by selecting the ‘Private’ box making it visible inside your My Media only.

Analytics

Will display comprehensive statistical information about the video. If the video you are currently looking at is a Video Quiz, then data about the quiz questions and student responses can be found here as well.

Launch Editor

This option will execute the video editor tool for the current video. With this tool you can do one of three things:

  1. You can edit the current video
  2. Your can turn the current Video into a Video Quiz (or edit the quiz if it is already one)
  3. You can insert hotspots into the video

Delete

*WARNING* deleting a video using this option will remove it from My Media, every Media Gallery and Content area currently embedded, as well as the Kaltura KMC. If you are not absolutely sure or want to remove a video from a specific course it is better to unpublish the video using the publish option instead. Once the video is deleted it cannot be recovered even by Kaltura.

Edit Media

Contains a multitude of options that will allow you to alter your video in a significant way. These options are specific to BCIT instance of Kaltura only:

  • Details
  • Publish
  • Options
  • Thumbnails
  • Downloads
  • Captions
  • Attachments
  • Timeline
  • Replace Media

Details

Will allow you to change the title, description and any searchable fields that are currently associated with the current video.

Publish

Will allow you to publish or unpublish the video from your course Media Galleries.

Options

Turn on/off commenting and video clipping options from students/viewers.

Thumbnails

Allows you to upload a custom thumbnail for your video (default thumbnail is the first frame at the 3rd second of a video).

Downloads

Turn on/off the ability for students/viewers to download the video.

Captions

Allows the captioning of videos by uploading either a .srt, .dxfp, .vtt or .txt file. Please note that machine captioning will only work on supported video players which can be configured on the Kaltura KMC. Please also note that BCIT’s Kaltura instance does not support professional captioning or caption ordering at this time.

Attachments

Allows you to upload attachments file such as powerpoints or word files to your video for students to download.

Timeline

Allows you to create chapters for your video. You can also insert slides (.jpg or .ppt files) into your video and have them display side by side using this option.

Replace Media

Allow you to completely replace a video with another video by uploading a file here

For more comprehensive information concerning any or all of the options above, please see Kaltura’s official documentation for Edit Media.
