Entries

The Entries sub-tool displays every single video that has been created at BCIT in the Learning Hub (and later Mediaspace). Inside Entries, you have the ability to alter metadata, captions & ownership rights of individual videos or groups of videos. You also have the ability to insert or remove videos from course Media Galleries in Brightspace as well. To do this go into view details > metadata, then click Browse Categories and under D2L > Site > Channel and select the Media Gallery.

For more detailed information concerning the Entries tab please see the Kaltura documentation

View Details

To view or change a video’s settings click on the video itself or click on the ellipses … > view details.

Metadata

The fields you see inside Metadata contribute to a video’s searchability inside both the KMC and in My Media/Media Gallery. The fields seen in here are same ones populated by the instructor in the Learning Hub when they first publish the video into their course on the Learning Hub.

Thumbnails

You can choose to upload or a replace a thumbnail for a video in this area.

Access Control

If the video is published outside of the Learning Hub (or BCIT MediaSpace) then this tool can be used to restrict which domains (websites) the video is allowed to appear in. These restrictions are created and managed using the Access Control Profile.

Scheduling

Can be used to restrict the window of time a video is viewable by. You can think of this a little like the start and end times for students that you see on the Learning Hub’s quiz tool, except for Kaltura videos

Flavors

Will let you restrict how Kaltura coverts a video file when it is first uploaded. (We will likely not need to touch this tool)

Distribution

Is used for publishing videos on platforms like YouTube with all of Kaltura’s metadata & flavors prepopulated. (We have no such integration and will not need to touch this tool)

Captions

This feature will let you upload, remove or replace caption files for videos. In order for captions to display properly, the video player playing videos must be properly configured in the STUDIO tab. The Learning Hub’s Video should however be properly configured at this point.

Related Files

This tool is used to upload files which can be downloaded by users.

Clips

This tool can be used to edit videos or add hotspots.

Users

This area is where you can specify or change the owner of the video as well as adding or changing any Co-Editors, Co-Publishers and Co-viewers.

  • Co-Editors - can edit the entry's details and metadata, trim media, replace media, edit captions, edit chapters and edit slides. Co-editors cannot delete media or add new co-editors and co-publishers. Co-editors can see the analytics page for the media they co-edit.
  • Co-Publishers - can publish media to their entitled Categories or Channels. This option must be enabled by your KMS administrator for this tab to display. Group support may be enabled for the Media Collaboration features. When enabled, you can select groups that may be assigned as co-editors/publishers for an entry.
  • Co-Viewers - are only able to view media and do not have editing permissions, nor are they allowed to view unlisted entries, unless they are also co-publishers or co-editors of that entry.
  • Owners - can do all of the above

Advertisements

This tool will not be applicable to ETS, it is used to create advertisements.

Sharing and Embedding

You can share or embed videos into different websites or documents using either the iframe code or preview url that is generated from this area. You can also change the preferred video player that this is displayed on using Select Player option at top.

Searching and Filtering

You can search for specific videos or groups of video using the Search Entries, Filter categories or Refine options.

Search entries

Enter plain text to filter videos by. You can also specify the type of metadata you want your text to match using the ⚙️ icon

Filter Categories

Will allow you to search for videos based on the categories they have associated to them. Videos that have been uploaded to the Learning Hub can be found under the following category structure: D2L > site > channel. (These are not CRN numbers)

Refine options

Will allow you to filter videos by a number of specific criteria that is available on the tool
