Playlists

Playlists created by Instructors in Learning Hub course Media Galleries can be viewed here. Unlike individual video in the Entries tool, you cannot assignment playlists a category. This means it is not possible to move or copy a playlist visible in a specific course media gallery and assign it to another course media gallery in the Learning Hub. It is only possible to provide the embed code to the instructor.
