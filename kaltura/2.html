Introduction to My Media

The My Media area is personal video repository that is unique to each user in the Learning Hub. It can be thought of as the area where all of your videos that are either created by you or that you have co-ownership of live.

Adding media to My Media

My Media has 4 options available to you for adding content, these are:

  1. Media Upload
  2. Express capture
  3. Kaltura Capture
  4. Video Quiz

Media Upload

Drag and drop a media file into the drop zone or click “Choose a file to upload” to select a media file (.mp4 format for video files is preferable).

Express Capture

Use this tool to quickly record yourself from your webcam. Clicking the red button will trigger a countdown which will let you know when the tool will start recording. Use the ⚙️ to change your camera and microphone settings.

Kaltura Capture

Kaltura Capture can be used to record your screen, yourself or both at the same time. This tool will also allow you to add annotations using the built in annotation tools while you are recording. To use Kaltura Capture, you must first install it on your personal computer as an app. We recommend reading the user guide for an in depth explanation on how this tool works.

Video Quiz

The Video Quiz option will take you to the video editing screen where you can make a copy of the video and then insert questions into it. This video can then be inserted into the Learning Hub’s Content area, have a grade item attached to it, and then be graded by the Learning Hub’s Grades tool. Video Quizzes can have four question types inserted into them, these are:

  1. Multiple Choice
  2. True/False
  3. Open Ended (not gradable)
  4. Reflection (not gradable)

Please see the Setting up a Video Quiz section for notes on how to enable a video quiz in a course

Publishing Videos from My Media

To publish your videos into your course Media Gallery from My Media, you must fill out all of the required fields when you first create your video. Once done, you can either tick the ‘Published box’ which will save your video into both My Media and Media gallery, or the ‘Private box’ which will save your video inside your My Media only

If you tick the Published box Kaltura’s API will detect all of the courses in the Learning Hub that you have instructor access to and will generate a list with all of these courses at the bottom. Tick the course(s) at the bottom to specify which course Media Gallery you want to add video into.

Searching/Filtering Videos

Use the search bar at the top or select ‘Filters’ and tick the options to search for videos that have specific criteria.
