Mediaspace Content Options

Just like the video options inside the Learning Hub, videos that are uploaded into Mediaspace have options which can be configured by a video’s owner/co-owner. Most of these options are quite similar to the ones found for videos in the Learning Hub, however there are some features exclusive to Mediaspace. Mediaspace content has the following options:

  • Details
  • Share
    • Link to Media Page
    • Embed
    • Email
  • Actions
    • Edit
    • Publish
    • Add to Playlist
    • Analytics
    • Add to Watch Later
    • Launch Editor
    • Delete

Details

Will show you a video’s title, tags and the names of any or all Channels or Galleries it is associated to.

Share

Will allow a video’s owner to share their videos via either a shared link, embed code or an email message containing the shared link.

Link to Media Page

Produces a URL which the video owner can send to others

Embed

Produces html code that the video owner can use to embed the video into other websites, web applications or web services.

Email

Opens up the owners preferred email and produces a URL in the body of the email which can be sent to others (Note: this feature will not work if the video has not been published into either a Channel or Gallery).

Link/Embed code security

Please note that security of a link for each option above is dependent entirely on where the video is published. For example if a video is published into a Gallery that is set to ‘Open’, then anyone will be able to access the video via generated link. In contrast if it is published in into a space that is ‘Private’, then only members of that Channel or Gallery may access it. Actions

Items under the Actions options will allow you to modify your video in a significant way.

Edit

Please see the Edit section for more details

Publish

Will allow you to publish the video into a Gallery or Channel if you have not done so upon creation. You can also unpublish (hide) the video from all Channels and Galleries by selecting the ‘Private’ box which will make the video visible only to you inside your My Media.

Analytics

Will display comprehensive statistical information about the video. If the video you are currently looking at is Video Quiz, then data about the quiz questions and responses can be found here as well.

Add to Watch Later

This option will pin the video to the ‘watch later’ playlist that you can see on the homepage.

Launch Editor

This option will execute the video editor tool. With this tool you can do one of three things:

  1. You can edit the video
  2. Your can turn the video into a Video Quiz (or edit the quiz if it is already one)
  3. You can insert hotspots into the video

Delete

*WARNING* deleting a video using this option will permanently remove it from your My Media as well as every Gallery and Channel that it is currently published in. If you are not absolutely 100 percent sure you want remove a video from Mediaspace, then we recommend unpublishing the video instead. Once a video is deleted it cannot be recovered, even by Kaltura.

Edit

Contains a multitude of options that will allow you to alter your video in a significant way. These options are:

  • Details
  • Publish
  • Options
  • Collaboration
  • Thumbnails
  • Downloads
  • Captions
  • Attachments
  • Timeline

Details

Will allow you to change the title, description and any searchable fields that are currently associated with the current video.

Publish

Will allow you to publish or unpublish the video into and from Channels and Galleries.

Options

Allows you to turn on/off commenting and viewer video clipping options.

Collaboration

Will allow you to switch ownership of a video with someone else or give someone else access to edit, publish or view a video

Media Owner

Will allow you to change a video’s owner allowing them to see it in their My Media and be given full ownership rights.

Media Collaborators

Will allow you to give one of three permissions to another person so that they may either edit, publish or view a video their my media

Co-editor

Will allow another user editing rights to a video

Co-publisher

Will allow another user to publish a video into a channel or gallery

Co-viewer

Will allow another user to view a video in their media gallery regardless if a video is published or not

Thumbnails

Allows you to upload a custom thumbnails for your video.

Downloads

Turn on/off the ability for viewers to download your video. Video options available for downloading that can be chosen are: the original source file that it was uploaded as, standard definition 480p, high definition 720p, and high definition 1080p.

Captions

Allows the captioning of videos by uploading either a .srt, .dxfp, .vtt or .txt file. Please note that all videos with audio will be captioned automatically using Kaltura machine captioning. Please also note that we do not support professional captioning or the caption ordering feature at this time.

Attachments

Allows you to upload attachment files such as .pptx, .docx, .xlsx, etc that your viewers can download.

Timeline

Allows you to create chapters (timestamps) or slides for your video. Slides can be in either .jpg, .png or .pptx format and will be displayed alongside your video.
