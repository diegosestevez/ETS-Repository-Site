Adding Kaltura Media in Content

Kaltura Media that has been published in either My Media or Media Gallery can be place inside the Content area of a course using Add Existing Activities.

Adding Kaltura Media to a Module

Click Add Existing Activities > Add Kaltura Media and then select </> Embed to automatically place the selected video inside a Module. You can click the ⚙️ to change the Embed options such as the Video Player.

Setting up a Video Quiz

To add a Video Quiz in Content you must first create one in My Media. Once this has been done add it like any other Kaltura Media with Add Existing Activities > Add Kaltura Media > Embed. Now click the Video Quiz and scroll to the bottom, in Activity Details > Assessment create a New Grade Item and Save it. Now go the Grades > Manage Grades and configure the points and weight (by default each MC and MS question on the video quiz will register as a point after the grade item is created).

Once configured, students only need to take the quiz and submit their answers in order for their marks to carry over to the grade item. Please note however that Reflection and Open Ended answer type questions CANNOT BE GRADED by Grades. You can see student answers to Open Ended questions by going to the original video either in My Media or Media Gallery (assuming you are the instructor who created it) clicking on Actions > Analytics > Quiz Questions or Quiz Users.
